# SEO Bandwagon

SEO tools and services without the corporate nonsense. Built in Port Townsend, WA since 2010.

## Local Development

Just open `index.html` in a browser. No build process needed.

## Deployment

This site is deployed to Netlify.

- **Staging:** seobandwagon.dev
- **Production:** seobandwagon.com

## Stack

- Pure HTML/CSS/JS (no framework)
- Fonts: Bebas Neue, Space Grotesk, Space Mono
- Hosting: Netlify

## Brand Colors

- Pink: `#F53796`
- Navy: `#000022`
- White: `#F5F5F5`
